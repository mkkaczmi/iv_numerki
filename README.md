# Numerical Methods Projects

This repository contains implementations of various numerical methods algorithms:

## Projects

1. **Nonlinear Equations Solver** (`nonlinear_equations_solver/`)
   - Implementation and comparison of two methods for finding roots of nonlinear equations
   - Methods: Bisection and Secant

2. **Linear Equations Solver** (`linear_equations_solver/`)
   - Implementation of Gaussian elimination for solving systems of linear equations
   - Features: File input, manual input, and dataset processing

3. **Newton Interpolation** (`newton_interpolation/`)
   - Implementation of Newton's interpolation method
   - Support for various function types and visualization

## Requirements

- Python 3.x
- NumPy
- Matplotlib

## Usage

Each project directory contains its own README.md with specific instructions for that project.
